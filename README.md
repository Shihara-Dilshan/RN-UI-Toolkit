# rn-ui-toolkit

RN-UI-Toolkit is to provide an all-in-one UI kit for creating apps in react native.

## Installation

```sh
npm install rn-ui-toolkit
```

## Usage

```js
import { multiply } from "rn-ui-toolkit";

// ...

const result = await multiply(3, 7);
```
